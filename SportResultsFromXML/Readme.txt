Import this project in your Eclipse/NetBeans IDE as an Existing Project into Workspace. You will find two java 
files (MatchList.java and MatchListSort.java) in “main/java” directory inside the “src” directory. “MatchList.java” 
is a main file. You will have two other directories (“main/resources” and “main/generatedFiles”) also inside the “src” 
directory. Data files are stored in the “resources” directory and generated files store in “generatedFiles” directory.
